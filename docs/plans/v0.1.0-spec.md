# NumPy-DL v0.1.0 — Neural Network Core

## Goal
Implement the foundational neural network components: Dense layer, activations (ReLU, Softmax), losses (MSE, Cross-Entropy), SGD optimizer, weight initialization, and an XOR training example.

## Architecture Decisions (from self-grill)
- Layer API: `forward(x)` caches input, `backward(grad)` returns gradient + updates params
- Model: Sequential wrapper + standalone layers
- Parameters: Typed attributes (`layer.W`, `layer.dW`)
- Losses: Pure functions returning `(loss, grad)`
- Activations: ReLU/Sigmoid as layers, Softmax forward-only in activations.py
- Init: Standalone functions (he_init, xavier_init) in utils/init.py

## Files to Create

### src/numpy_dl/nn/layers.py
- `Layer` base class with `forward()` and `backward()` abstract methods
- `Dense(in_features, out_features, weight_init=he_init)` layer
  - Stores: W (weight matrix), b (bias vector), input (cached), dW, db
  - Forward: `output = x @ W + b`
  - Backward: computes dW, db, returns `grad_output @ W.T`
- Creator references: Rosenblatt (1958) for perceptron, Rumelhart et al. (1986) for backprop
- Math equations: forward (y = Wx + b), backward (dW = x^T * dL/dy, db = sum(dL/dy))

### src/numpy_dl/nn/activations.py
- `ReLU` layer: `forward(x) = max(0, x)`, `backward(grad) = grad * (input > 0)`
- `Sigmoid` layer: `forward(x) = 1/(1+exp(-x))`, `backward(grad) = grad * output * (1-output)`
- `softmax(x)` function (forward-only): `softmax(x_i) = exp(x_i) / sum(exp(x_j))`
- Creator references: Hahnloser (1998) for ReLU, Bridle (1990) for softmax

### src/numpy_dl/nn/losses.py
- `mse_loss(y_pred, y_true)` → `(loss, grad)` where `grad = 2*(y_pred - y_true)/n`
- `softmax_cross_entropy_loss(y_pred, y_true)` → `(loss, grad)` where `grad = (softmax(y_pred) - y_true)/n`
- Creator references: Legendre (1805) for MSE, Shannon (1948) for cross-entropy

### src/numpy_dl/optim/sgd.py
- `SGD(params, lr=0.01)` class
  - `step()`: for each (param, grad) pair, `param -= lr * grad`
- Creator references: Robbins & Monro (1951)

### src/numpy_dl/utils/init.py
- `he_init(shape)` → `randn(shape) * sqrt(2/fan_in)` (He et al., 2015)
- `xavier_init(shape)` → `randn(shape) * sqrt(1/fan_in)` (Glorot & Bengio, 2010)

### examples/xor_sgd.py
- Train XOR using Dense(2,4) + ReLU + Dense(4,1) + MSE + SGD
- Full training loop visible, prints loss every 100 epochs
- Should converge to loss < 0.01 in ~1000 epochs
