# NumPy-DL v0.4.0 — DQN with Replay Buffer and Target Network

## Goal
Implement Deep Q-Network (DQN) with experience replay and target network. Add GymAdapter for real Gymnasium environments.

## Files to Create

### src/numpy_dl/memory/replay_buffer.py
- Simple circular buffer storing (state, action, reward, next_state, done) tuples
- push(state, action, reward, next_state, done)
- sample(batch_size) → returns numpy arrays
- __len__() → current size
- Creator references: Lin (1993) "Reinforcement learning with experience replay"

### src/numpy_rl/utils.py (in rl/)
- epsilon_greedy(q_values, epsilon, n_actions) → action
- Creator references: Sutton & Barto (2018)

### src/numpy_rl/dqn.py (in rl/)
- DQNAgent class:
  - __init__(state_dim, action_dim, hidden_dim=128, lr=1e-3, gamma=0.99, epsilon=1.0, ...)
  - q_net: Sequential(Dense→ReLU→Dense→ReLU→Dense)
  - target_net: copy of q_net (updated periodically)
  - select_action(state): ε-greedy using q_net
  - store_transition(s, a, r, s2, done): push to replay buffer
  - train_step(batch_size=32): sample batch, compute MSE loss, update q_net
  - update_target(): copy q_net weights to target_net
  - train(env, episodes): full training loop
- Creator references: Mnih et al. (2015) "Human-level control through deep reinforcement learning"
- Math: L = E[(r + γ max_a' Q_target(s',a') - Q(s,a))^2]

### src/numpy_dl/environments/gym_adapter.py
- GymAdapter wrapper: converts Gymnasium API to numpy-dl minimal API
- reset() → state (discards info dict)
- step(action) → (state, reward, done, info) (merges terminated+truncated)

### examples/dqn_cartpole.py
- Train DQN on CartPole-v1
- Print episode rewards, target network sync every 10 episodes

### tests/rl/test_dqn.py
- Test DQN agent initialization
- Test select_action returns valid action
- Test train_step runs without error
- Test target network update

### tests/memory/test_replay_buffer.py
- Test push and sample
- Test capacity overflow
- Test batch shapes

## Constraints
- CartPole requires gymnasium (optional dependency)
- Handle case where gymnasium not installed (skip tests gracefully)
