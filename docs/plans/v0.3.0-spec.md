# NumPy-DL v0.3.0 — Tabular Q-Learning + Environments

## Goal
Implement tabular Q-Learning with GridWorld and FrozenLake environments.

## Files to Create

### src/numpy_dl/environments/grid_world.py
- GridWorld environment: n×n grid, agent finds goal
- Actions: up, right, down, left (0-3)
- Reward: -1 per step, +10 for reaching goal
- Creator references: Sutton & Barto (2018) "Reinforcement Learning: An Introduction"
- Math: Q(s,a) ← Q(s,a) + α[r + γ max_a' Q(s',a') - Q(s,a)]

### src/numpy_dl/environments/frozen_lake.py
- FrozenLake: 4×4 grid with holes, frozen tiles, goal
- Discrete states (16), discrete actions (4)
- Stochastic transitions (slippery)
- Creator references: OpenAI Gymnasium

### src/numpy_rl/q_learning.py (in rl/)
- QLearningAgent class:
  - Q-table: np.ndarray (n_states, n_actions)
  - select_action(state, epsilon): ε-greedy
  - update(state, action, reward, next_state, done): Q-learning update
  - train(env, episodes): full training loop
- Creator references: Watkins & Dayan (1992) "Q-Learning"
- Math: Q(s,a) ← Q(s,a) + α[r + γ max_a' Q(s',a') - Q(s,a)]

### examples/gridworld_q_learning.py
- Train Q-learning agent on GridWorld
- Print rewards per episode

### examples/frozen_lake_q_learning.py
- Train Q-learning agent on FrozenLake
