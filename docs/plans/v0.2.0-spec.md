# NumPy-DL v0.2.0 — Adam Optimizer + Save/Load Weights

## Goal
Implement Adam optimizer and weight save/load utilities. Improve model API.

## Files to Create

### src/numpy_dl/optim/adam.py
- Adam optimizer class with:
  - First moment (m) and second moment (v) estimates per parameter
  - Bias correction: m_hat = m / (1 - β1^t), v_hat = v / (1 - β2^t)
  - Update: θ = θ - lr * m_hat / (sqrt(v_hat) + ε)
- Creator references: Kingma & Ba (2014) "Adam: A Method for Stochastic Optimization"
- Math equations:
  - m_t = β1 * m_{t-1} + (1 - β1) * g_t
  - v_t = β2 * v_{t-1} + (1 - β2) * g_t^2
  - m_hat = m_t / (1 - β1^t)
  - v_hat = v_t / (1 - β2^t)
  - θ_t = θ_{t-1} - lr * m_hat / (sqrt(v_hat) + ε)
- Defaults: lr=0.001, β1=0.9, β2=0.999, ε=1e-8

### src/numpy_dl/utils/save_load.py
- save_weights(model, path) — saves to .npz with indexed keys
- load_weights(model, path) — loads from .npz
- Works with Sequential and standalone layers
- Creator references: NumPy documentation
